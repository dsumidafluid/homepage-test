Rhinoceros 有用操作メモ

Rhinoでモデルを整理しながら作るための基本操作を、一般的な作業メモとしてまとめます。公式ヘルプの機能名を参考にしつつ、本文は独自に整理しています。

Model

モデルを整理する

レイヤを先に分ける

部品、補助線、寸法、参照、出力用などでレイヤを分けると、表示切替と選択が楽になります。

例:
00_reference
10_parts
20_curves
90_dimensions

名前を付ける

重要なオブジェクトやブロックには名前を付けます。後で検索・選択しやすくなります。

命名例:
base_plate
guide_curve_A
section_profile_01

ブロックで繰り返し部品を管理

同じ形状を何度も置く場合はブロック化します。修正箇所を減らし、モデルの一貫性を保ちやすくします。

向いているもの:
ボルト、同形パネル、記号、標準部品

単位と許容差を確認

作業前に単位、絶対許容差、角度許容差を確認します。後工程で寸法が合わない事故を減らします。

確認:
単位
絶対許容差
角度許容差

View

表示と選択を使い分ける

表示モードを切り替える

ワイヤ、シェーディング、レンダー系表示を目的別に使います。形状確認と見栄え確認を分けるのがコツです。

作業中: ワイヤ/シェーディング
確認: レンダー系
説明: 見やすい表示設定

Named Viewを使う

よく見る方向や説明用ビューを保存しておくと、比較や出力時に同じ見え方を再現できます。

例:
front_detail
isometric_overview
section_check

選択フィルターを意識する

点、曲線、サーフェス、ポリサーフェスなど、対象を絞ると誤選択が減ります。

選択前に考える:
何を選びたいか
何を選びたくないか

一時的に隠す

作業の邪魔になる部品は削除せず非表示にします。確認後に戻せるようにするのが安全です。

Hide / Show
Isolate系操作
レイヤ表示切替

Edit

編集時の事故を減らす

補助線レイヤを作る

作図用の線と最終形状を混ぜないようにします。不要になった補助線をまとめて非表示・削除できます。

20_construction
21_section_curves
22_reference_points

編集前に複製を残す

大きく形状を変える前は、別レイヤや別ファイルに退避します。戻せる安心感が作業速度を上げます。

backup_before_trim
backup_before_boolean

Booleanは結果を検査する

Boolean後は穴、余分な面、開いたエッジがないか確認します。失敗時は分割・トリムで手作業に戻ります。

確認:
エッジ
法線
隙間
不要面

履歴が必要な作業を分ける

後から調整したい形状は、元の曲線や断面を残します。完成形だけ残すと修正に弱くなります。

残すもの:
元曲線
断面
基準点
作業メモ

Output

出力前チェック

寸法と注釈を別レイヤへ

モデル形状と注釈を分けると、出力時の表示制御が簡単です。

90_dimensions
91_annotations
92_layout_notes

レイアウトで紙面確認

説明資料や図面化では、モデル空間だけでなくレイアウト上で尺度と見え方を確認します。

確認:
尺度
線の太さ
寸法
余白

書き出し形式を目的で選ぶ

受け渡し先に合わせて、3DM、DWG、DXF、STEP、IGESなどを選びます。単位と尺度の確認もセットです。

形状共有: 3DM/STEP
2D共有: DWG/DXF
資料: PDF/画像
