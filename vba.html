Excel VBA Dictionary

Excel VBA 有用ルーチン集

よく使う処理を、小さな関数・Subとして再利用しやすい形で整理します。 業務固有の内容は含めず、一般的なサンプルとして使える形にしています。

Basics

ブック・シート・セル操作

最終行を取得する

指定列の最後に値がある行番号を返します。

Function LastRow(ws As Worksheet, Optional col As Long = 1) As Long
    LastRow = ws.Cells(ws.Rows.Count, col).End(xlUp).Row
End Function

最終列を取得する

指定行の右端に値がある列番号を返します。

Function LastCol(ws As Worksheet, Optional rowNo As Long = 1) As Long
    LastCol = ws.Cells(rowNo, ws.Columns.Count).End(xlToLeft).Column
End Function

シート存在チェック

同じ名前のシートがあるか安全に判定します。

Function SheetExists(sheetName As String, Optional wb As Workbook) As Boolean
    Dim ws As Worksheet
    If wb Is Nothing Then Set wb = ThisWorkbook

    On Error Resume Next
    Set ws = wb.Worksheets(sheetName)
    On Error GoTo 0

    SheetExists = Not ws Is Nothing
End Function

シートを取得または作成

なければ新規作成して返します。

Function GetOrCreateSheet(sheetName As String, Optional wb As Workbook) As Worksheet
    If wb Is Nothing Then Set wb = ThisWorkbook

    If SheetExists(sheetName, wb) Then
        Set GetOrCreateSheet = wb.Worksheets(sheetName)
    Else
        Set GetOrCreateSheet = wb.Worksheets.Add(After:=wb.Worksheets(wb.Worksheets.Count))
        GetOrCreateSheet.Name = sheetName
    End If
End Function

画面更新を止めて高速化

重い処理の前後で使う基本形です。

Sub RunFast()
    On Error GoTo CleanUp
    Application.ScreenUpdating = False
    Application.EnableEvents = False
    Application.Calculation = xlCalculationManual

    ' ここに重い処理を書く

CleanUp:
    Application.Calculation = xlCalculationAutomatic
    Application.EnableEvents = True
    Application.ScreenUpdating = True
End Sub

範囲を配列で読む

セルを1つずつ読むより高速です。

Sub ReadRangeToArray()
    Dim ws As Worksheet
    Dim values As Variant

    Set ws = ThisWorkbook.Worksheets("Sheet1")
    values = ws.Range("A1").CurrentRegion.Value

    Debug.Print values(1, 1)
End Sub

Files

ファイル・フォルダ処理

フォルダを作成する

存在しない場合だけフォルダを作ります。

Sub EnsureFolder(folderPath As String)
    If Len(Dir(folderPath, vbDirectory)) = 0 Then
        MkDir folderPath
    End If
End Sub

ファイル存在チェック

指定パスにファイルがあるか確認します。

Function FileExists(filePath As String) As Boolean
    FileExists = (Len(Dir(filePath)) > 0)
End Function

フォルダ内のExcelファイルを処理

拡張子を指定して順番に処理します。

Sub LoopExcelFiles(folderPath As String)
    Dim fileName As String

    If Right(folderPath, 1) <> "\" Then folderPath = folderPath & "\"
    fileName = Dir(folderPath & "*.xlsx")

    Do While fileName <> ""
        Debug.Print folderPath & fileName
        fileName = Dir()
    Loop
End Sub

CSVとして保存する

指定シートを一時ブックにコピーしてCSV出力します。

Sub SaveSheetAsCsv(ws As Worksheet, csvPath As String)
    ws.Copy
    With ActiveWorkbook
        Application.DisplayAlerts = False
        .SaveAs Filename:=csvPath, FileFormat:=xlCSVUTF8
        .Close SaveChanges:=False
        Application.DisplayAlerts = True
    End With
End Sub

テキストファイルを一括読み込み

小さめのログや設定ファイルを読むときの基本形です。

Function ReadTextFile(filePath As String) As String
    Dim fileNo As Integer

    fileNo = FreeFile
    Open filePath For Input As #fileNo
    ReadTextFile = Input$(LOF(fileNo), fileNo)
    Close #fileNo
End Function

タイムスタンプ付きファイル名

出力ファイルの上書きを避けたいときに便利です。

Function TimestampName(prefix As String, ext As String) As String
    TimestampName = prefix & "_" & Format(Now, "yyyymmdd_hhnnss") & "." & ext
End Function

CSV

CSV入力・出力・形式整形

CSVをUTF-8で開く

Workbooks.OpenTextを使い、カンマ区切りとして開きます。文字化け対策として文字コード指定を意識します。

Sub OpenCsvUtf8(csvPath As String)
    Workbooks.OpenText _
        Filename:=csvPath, _
        Origin:=65001, _
        DataType:=xlDelimited, _
        Comma:=True
End Sub

シートをCSV UTF-8で保存

対象シートだけを一時ブックにコピーし、CSVとして保存します。

Sub ExportSheetToCsvUtf8(ws As Worksheet, csvPath As String)
    ws.Copy
    With ActiveWorkbook
        Application.DisplayAlerts = False
        .SaveAs Filename:=csvPath, FileFormat:=xlCSVUTF8
        .Close SaveChanges:=False
        Application.DisplayAlerts = True
    End With
End Sub

CSV用に値をエスケープ

カンマ、改行、ダブルクォートを含む値を安全に出力します。

Function CsvEscape(value As Variant) As String
    Dim text As String
    text = CStr(value)
    text = Replace(text, """", """""")

    If InStr(text, ",") > 0 Or InStr(text, vbLf) > 0 Or InStr(text, """") > 0 Then
        CsvEscape = """" & text & """"
    Else
        CsvEscape = text
    End If
End Function

範囲をCSV文字列へ変換

任意のRangeをCSV形式の文字列にします。

Function RangeToCsvText(target As Range) As String
    Dim r As Long, c As Long
    Dim line As String, result As String

    For r = 1 To target.Rows.Count
        line = ""
        For c = 1 To target.Columns.Count
            If c > 1 Then line = line & ","
            line = line & CsvEscape(target.Cells(r, c).Value)
        Next c
        result = result & line & vbCrLf
    Next r

    RangeToCsvText = result
End Function

CSV出力前に列形式を整える

日付やIDが勝手に変換されないよう、必要な列は文字列形式にしてから値を書きます。

Sub FormatCsvColumns(ws As Worksheet)
    ws.Columns("A").NumberFormat = "@"
    ws.Columns("B").NumberFormat = "yyyy-mm-dd"
    ws.Columns("C").NumberFormat = "0.00"
End Sub

区切り文字と余分な空白を整える

取り込んだCSVの前後空白や全角スペースを簡単に整えます。

Function NormalizeCsvField(value As Variant) As String
    Dim text As String
    text = CStr(value)
    text = Replace(text, "　", " ")
    NormalizeCsvField = Trim(text)
End Function

Data

データ整形・集計

Dictionaryで件数集計

A列の値ごとに出現回数を数えます。

Sub CountByDictionary()
    Dim dict As Object
    Dim ws As Worksheet
    Dim i As Long, key As String

    Set dict = CreateObject("Scripting.Dictionary")
    Set ws = ThisWorkbook.Worksheets("Sheet1")

    For i = 2 To LastRow(ws, 1)
        key = CStr(ws.Cells(i, 1).Value)
        If Len(key) > 0 Then
            If Not dict.Exists(key) Then dict.Add key, 0
            dict(key) = dict(key) + 1
        End If
    Next i
End Sub

重複行を削除する

指定列をキーに重複を除外します。

Sub RemoveDuplicateRows(ws As Worksheet, keyColumn As Long)
    Dim last As Long
    last = LastRow(ws, keyColumn)

    ws.Range(ws.Cells(1, 1), ws.Cells(last, LastCol(ws, 1))) _
        .RemoveDuplicates Columns:=keyColumn, Header:=xlYes
End Sub

空白を除去して文字列化

セル値を比較しやすく整えます。

Function CleanText(value As Variant) As String
    CleanText = Trim(Replace(CStr(value), vbCrLf, " "))
End Function

列番号を列記号へ変換

1をA、27をAAに変換します。

Function ColumnLetter(colNo As Long) As String
    ColumnLetter = Split(Cells(1, colNo).Address(False, False), "$")(0)
End Function

指定文字で分割して縦に出力

カンマ区切りの値を一覧化します。

Sub SplitToRows(sourceText As String, outputCell As Range, Optional delimiter As String = ",")
    Dim parts As Variant
    Dim i As Long

    parts = Split(sourceText, delimiter)
    For i = LBound(parts) To UBound(parts)
        outputCell.Offset(i, 0).Value = Trim(parts(i))
    Next i
End Sub

フィルター後の可視セルだけ処理

抽出済みデータだけを対象にします。

Sub LoopVisibleRows(targetRange As Range)
    Dim cell As Range

    On Error Resume Next
    For Each cell In targetRange.SpecialCells(xlCellTypeVisible)
        Debug.Print cell.Value
    Next cell
    On Error GoTo 0
End Sub

Error Handling

エラー処理・確認用ルーチン

標準的なエラー処理

どこで失敗したかをメッセージに出します。

Sub SampleWithErrorHandler()
    On Error GoTo ErrHandler

    ' ここに処理を書く
    Exit Sub

ErrHandler:
    MsgBox "エラー番号: " & Err.Number & vbCrLf & Err.Description, vbExclamation
End Sub

Sub MeasureTime()

    Debug.Print "処理時間: " & Format(Timer - startTime, "0.00") & " 秒"
End Sub

選択範囲を確認する

マクロ実行前の誤操作を減らします。

Function ConfirmSelection() As Boolean
    ConfirmSelection = (MsgBox(Selection.Address(False, False) & " を処理しますか？", _
        vbYesNo + vbQuestion) = vbYes)
End Function
