VBA Debug Setup

Excel VBA デバッグ環境の設定

VBAはデバッグ環境を少し整えるだけで、原因調査がかなり楽になります。 まずは「必ず宣言」「止める場所を作る」「変数を見る」「ログを残す」の4点を押さえます。

Settings

最初に整えるVBE設定

Option Explicitを自動挿入する

VBEで「ツール」→「オプション」→「編集」→「変数の宣言を強制する」をオンにします。新しいモジュールに自動でOption Explicitが入ります。

Option Explicit

自動構文チェックを好みで調整

入力途中の警告が邪魔なら「自動構文チェック」をオフにします。赤字表示は残るので、エラー位置は後から確認できます。

' ツール → オプション → 編集
' 自動構文チェック: 好みでON/OFF

エラートラップは「クラスモジュールで中断」

通常は「クラスモジュールで中断」が扱いやすいです。想定外の場所で止まり、標準的なエラー処理とも相性がよい設定です。

' ツール → オプション → 全般
' エラートラップ: クラスモジュールで中断

よく使うウィンドウを表示

プロジェクト、プロパティ、イミディエイト、ローカル、ウォッチを使える状態にします。表示メニューから開けます。

' Ctrl + G: イミディエイト
' 表示 → ローカル ウィンドウ
' 表示 → ウォッチ ウィンドウ

Break

処理を止めて確認する

ブレークポイント

止めたい行にカーソルを置いてF9。実行時にその行で止まり、変数やセルの状態を確認できます。

Sub Sample()
    Dim total As Long
    total = 10  ' ここでF9
    Debug.Print total
End Sub

Stop文で必ず止める

一時的な確認用です。公開・配布前には消します。

Sub CheckPoint()
    Stop
    ' ここで必ず中断する
End Sub

ステップ実行

F8で1行ずつ実行します。処理の流れを追うときに最も使う操作です。

' F8: ステップイン
' Shift + F8: ステップオーバー
' Ctrl + Shift + F8: ステップアウト

カーソル行まで実行

止まりたい行にカーソルを置いてCtrl+F8。長い処理の途中まで一気に進めたいときに便利です。

' Ctrl + F8: カーソル行の前まで実行

Inspect

変数・条件・オブジェクトを見る

イミディエイトで値を見る

Ctrl+Gで開き、? に続けて式を書くと、その場で値を確認できます。

? ActiveSheet.Name
? Range("A1").Value
? Now

Debug.Printでログを出す

実行中の状態をイミディエイトウィンドウに出します。

Debug.Print "row=", i, "value=", ws.Cells(i, 1).Value

ローカルウィンドウ

現在のSub/Function内の変数を一覧で見られます。配列やオブジェクトの中身を追うときに便利です。

' 表示 → ローカル ウィンドウ

ウォッチ式

特定の変数や条件を監視します。「値が変わったら中断」や「Trueになったら中断」に使えます。

' 例: i > 1000
' 例: total < 0
' 例: dict.Exists(key)

Logging

調査しやすいコードの形

処理入口と出口をログに出す

どこまで進んだかを簡単に追えるようにします。

Sub Main()
    Debug.Print Now, "Main start"
    ' 処理
    Debug.Print Now, "Main end"
End Sub

Err.NumberとErr.Descriptionを残す

エラーが出たときに原因調査しやすくします。

Sub RunWithErrorLog()
    On Error GoTo ErrHandler

    ' 処理
    Exit Sub

ErrHandler:
    Debug.Print "Error", Err.Number, Err.Description
    MsgBox Err.Description, vbExclamation
End Sub

小さいSubに分ける

1つの巨大マクロより、入力、変換、出力を分ける方が止めやすく調べやすいです。

Sub Main()
    ReadInput
    TransformData
    WriteOutput
End Sub

確認用フラグを用意する

調査中だけ詳細ログを出せるようにします。

Const DEBUG_MODE As Boolean = True

If DEBUG_MODE Then
    Debug.Print "count=", itemCount
End If
