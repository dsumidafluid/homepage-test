Numerical Methods

主要な数値解析アルゴリズム

数値解析の代表的な計算手法を、目的、使いどころ、注意点、疑似コードで整理します。 まずは実装前に流れを思い出すための逆引きメモです。

Root Finding

方程式の解を求める

二分法

区間[a, b]でf(a)とf(b)の符号が異なるとき、解を安全に挟み込みます。遅いが安定しやすい方法です。

while (b - a > tolerance) {
    c = (a + b) / 2
    if f(a) * f(c) <= 0:
        b = c
    else:
        a = c
}
root = (a + b) / 2

ニュートン法

初期値から接線を使って高速に解へ近づきます。導関数が必要で、初期値が悪いと発散することがあります。

x = initial
for i in 1..maxIteration:
    dx = f(x) / df(x)
    x = x - dx
    if abs(dx) < tolerance:
        break
root = x

割線法

ニュートン法の導関数を差分で近似する方法です。導関数を明示できないときに使いやすいです。

x0 = first
x1 = second
for i in 1..maxIteration:
    x2 = x1 - f(x1) * (x1 - x0) / (f(x1) - f(x0))
    if abs(x2 - x1) < tolerance:
        break
    x0 = x1
    x1 = x2

固定点反復法

x = g(x)の形に変形して反復します。収束条件を満たす形に変形できるかが重要です。

x = initial
for i in 1..maxIteration:
    next = g(x)
    if abs(next - x) < tolerance:
        break
    x = next

Integration

数値積分

台形則

区間を小さな台形に分けて面積を足します。実装が簡単で、滑らかな関数なら分割数を増やすと精度が上がります。

h = (b - a) / n
sum = (f(a) + f(b)) / 2
for i in 1..n-1:
    x = a + i * h
    sum += f(x)
integral = h * sum

シンプソン則

2次曲線で近似する積分法です。分割数nは偶数にします。台形則より少ない分割で高精度になりやすいです。

h = (b - a) / n
sum = f(a) + f(b)
for i in 1..n-1:
    x = a + i * h
    if i is odd:
        sum += 4 * f(x)
    else:
        sum += 2 * f(x)
integral = h * sum / 3

中点則

各小区間の中央の値で面積を近似します。単純で、関数評価点を端点からずらしたい場合に使えます。

h = (b - a) / n
sum = 0
for i in 0..n-1:
    x = a + (i + 0.5) * h
    sum += f(x)
integral = h * sum

モンテカルロ積分

乱数で平均値を推定します。高次元積分で使われますが、結果にばらつきがあり収束は遅めです。

sum = 0
for i in 1..samples:
    x = random(a, b)
    sum += f(x)
integral = (b - a) * sum / samples

ODE

常微分方程式

オイラー法

傾きdy/dx=f(x,y)を使って次の点へ進みます。実装は簡単ですが、刻み幅が大きいと誤差が増えます。

x = x0
y = y0
while x < end:
    y = y + h * f(x, y)
    x = x + h

改良オイラー法

予測した傾きと次点の傾きを平均します。オイラー法より精度が上がりやすいです。

k1 = f(x, y)
yPredict = y + h * k1
k2 = f(x + h, yPredict)
y = y + h * (k1 + k2) / 2
x = x + h

4次のルンゲクッタ法

代表的な高精度手法です。計算量は増えますが、実用上よく使われます。

k1 = f(x, y)
k2 = f(x + h/2, y + h*k1/2)
k3 = f(x + h/2, y + h*k2/2)
k4 = f(x + h, y + h*k3)
y = y + h * (k1 + 2*k2 + 2*k3 + k4) / 6
x = x + h

Linear Algebra

連立一次方程式・反復法

ガウス消去法

行基本変形で上三角行列にしてから後退代入します。小規模な連立方程式の基本です。

for k in 0..n-2:
    choose pivot row
    eliminate rows below pivot

for i in n-1..0:
    x[i] = (b[i] - sum(A[i][j] * x[j])) / A[i][i]

ヤコビ法

前回の全成分を使って次の近似値を計算します。実装が単純で並列化しやすい反復法です。

repeat until convergence:
    for i in 0..n-1:
        next[i] = (b[i] - sum(A[i][j] * x[j], j != i)) / A[i][i]
    x = next

ガウス・ザイデル法

更新済みの成分をすぐ次の計算に使います。ヤコビ法より速く収束することがあります。

repeat until convergence:
    for i in 0..n-1:
        x[i] = (b[i] - sum(A[i][j] * x[j], j != i)) / A[i][i]

収束判定

反復計算では、更新量や残差が十分小さくなったかを見ます。最大反復回数も必ず設定します。

error = norm(next - current)
if error < tolerance:
    converged = true
if iteration > maxIteration:
    stop as not converged
