Java Dictionary

Java 有用ルーチン集

Javaで何度も書く処理を、小さなメソッドや短いサンプルとして整理します。 まずは標準ライブラリ中心に、仕事固有の情報を含まない一般的な部品としてまとめています。

Strings

文字列処理

nullまたは空文字を判定

入力値チェックでよく使う基本形です。

public static boolean isBlank(String value) {
    return value == null || value.trim().isEmpty();
}

デフォルト値を返す

nullや空文字のときに代替文字列を返します。

public static String defaultIfBlank(String value, String defaultValue) {
    return isBlank(value) ? defaultValue : value;
}

文字列を安全に数値化

変換できない場合はOptional.empty()を返します。

public static Optional<Integer> parseIntSafe(String value) {
    try {
        return Optional.of(Integer.parseInt(value.trim()));
    } catch (Exception e) {
        return Optional.empty();
    }
}

区切り文字で結合

nullを除外して読みやすく結合します。

public static String joinNonNull(List<String> values, String delimiter) {
    return values.stream()
            .filter(Objects::nonNull)
            .collect(Collectors.joining(delimiter));
}

先頭だけ大文字にする

ラベルや表示名の整形に使えます。

public static String capitalize(String value) {
    if (isBlank(value)) return value;
    return value.substring(0, 1).toUpperCase() + value.substring(1);
}

固定長で左ゼロ埋め

IDや番号の表示整形に使います。

public static String zeroPad(int number, int length) {
    return String.format("%0" + length + "d", number);
}

Collections

List・Map・Stream

Listを安全に空リスト化

nullチェックを呼び出し側に散らさないための部品です。

public static <T> List<T> emptyIfNull(List<T> values) {
    return values == null ? Collections.emptyList() : values;
}

重複を除去する

順序を保ったまま重複を取り除きます。

public static <T> List<T> distinctKeepOrder(List<T> values) {
    return new ArrayList<>(new LinkedHashSet<>(values));
}

Listを指定サイズで分割

大量データをバッチ処理するときに便利です。

public static <T> List<List<T>> chunk(List<T> values, int size) {
    List<List<T>> result = new ArrayList<>();
    for (int i = 0; i < values.size(); i += size) {
        result.add(values.subList(i, Math.min(i + size, values.size())));
    }
    return result;
}

Mapの値を安全に取得

キーがない場合のデフォルトを指定します。

public static <K, V> V getOrDefault(Map<K, V> map, K key, V defaultValue) {
    if (map == null) return defaultValue;
    return map.getOrDefault(key, defaultValue);
}

要素の件数を集計する

文字列ごとの出現回数をMapにします。

public static Map<String, Long> countByValue(List<String> values) {
    return values.stream()
            .collect(Collectors.groupingBy(Function.identity(), LinkedHashMap::new, Collectors.counting()));
}

条件に合う最初の要素を取得

見つからない場合はOptional.empty()です。

public static <T> Optional<T> findFirst(List<T> values, Predicate<T> predicate) {
    return values.stream()
            .filter(predicate)
            .findFirst();
}

Files

ファイル・CSV・パス処理

テキストファイルを全行読む

UTF-8のテキストをListとして読み込みます。

public static List<String> readLines(Path path) throws IOException {
    return Files.readAllLines(path, StandardCharsets.UTF_8);
}

テキストファイルへ書き込む

親フォルダを作ってから保存します。

public static void writeText(Path path, String text) throws IOException {
    Files.createDirectories(path.getParent());
    Files.writeString(path, text, StandardCharsets.UTF_8);
}

フォルダ内のファイル一覧

拡張子で絞り込んでPathの一覧を返します。

public static List<Path> listFiles(Path folder, String extension) throws IOException {
    try (Stream<Path> stream = Files.list(folder)) {
        return stream
                .filter(Files::isRegularFile)
                .filter(path -> path.toString().endsWith(extension))
                .toList();
    }
}

CSVの1行を分割する簡易版

引用符を扱わない単純なCSV向けです。

public static List<String> splitCsvSimple(String line) {
    return Arrays.stream(line.split(",", -1))
            .map(String::trim)
            .toList();
}

ファイル名にタイムスタンプを付ける

ログや出力ファイルの上書きを避けます。

public static String timestampFileName(String prefix, String extension) {
    String stamp = LocalDateTime.now().format(DateTimeFormatter.ofPattern("yyyyMMdd_HHmmss"));
    return prefix + "_" + stamp + "." + extension;
}

ファイルサイズを読みやすく表示

バイト数をKB/MB表記にします。

public static String humanSize(long bytes) {
    if (bytes < 1024) return bytes + " B";
    if (bytes < 1024 * 1024) return String.format("%.1f KB", bytes / 1024.0);
    return String.format("%.1f MB", bytes / 1024.0 / 1024.0);
}

Date Time

日付・例外・ログ

日付文字列をLocalDateへ変換

形式が合わない場合はOptional.empty()にします。

public static Optional<LocalDate> parseDate(String value, String pattern) {
    try {
        return Optional.of(LocalDate.parse(value, DateTimeFormatter.ofPattern(pattern)));
    } catch (DateTimeParseException e) {
        return Optional.empty();
    }
}

処理時間を測る

簡単な性能確認用の定番コードです。

public static void measure(String name, Runnable task) {
    long start = System.nanoTime();
    task.run();
    long elapsedMs = (System.nanoTime() - start) / 1_000_000;
    System.out.println(name + ": " + elapsedMs + " ms");
}

例外をメッセージ付きで投げる

前提条件を明確にしたいときに使います。

public static void require(boolean condition, String message) {
    if (!condition) {
        throw new IllegalArgumentException(message);
    }
}

try-with-resourcesの基本形

閉じ忘れを防ぐファイル読み込みの形です。

public static void printFile(Path path) throws IOException {
    try (BufferedReader reader = Files.newBufferedReader(path, StandardCharsets.UTF_8)) {
        String line;
        while ((line = reader.readLine()) != null) {
            System.out.println(line);
        }
    }
}

Optionalを値に変換

空の場合に例外ではなくデフォルト値を返します。

public static <T> T orElse(Optional<T> optional, T defaultValue) {
    return optional.orElse(defaultValue);
}

簡易ログ出力

学習用や小さなツール向けの最小ログです。

public static void log(String message) {
    String now = LocalDateTime.now().format(DateTimeFormatter.ISO_LOCAL_DATE_TIME);
    System.out.println("[" + now + "] " + message);
}
