Access Notes

Access 基礎・有用ルーチン

小規模なデータ管理で使うAccessの基本構成と、CSV入出力、クエリ実行、レコードセット処理の定番ルーチンをまとめます。

Basics

Accessの基本部品

テーブル

データを保存する場所です。主キー、データ型、必須入力、インデックスを先に決めると崩れにくくなります。

例:
T_顧客
- 顧客ID: オートナンバー
- 顧客名: 短いテキスト
- 登録日: 日付/時刻

クエリ

抽出、並べ替え、集計、更新を行う部品です。直接テーブルを触る前に、選択クエリで結果を確認します。

SELECT 顧客ID, 顧客名
FROM T_顧客
ORDER BY 顧客ID;

フォーム

入力画面です。ユーザーに直接テーブルを見せず、フォーム経由で入力させるとミスを減らせます。

フォームの役割:
- 入力チェック
- 表示項目の整理
- ボタン操作の配置

レポート

印刷やPDF化向けの出力画面です。見積書、一覧表、集計表などの定型出力に向いています。

レポートの流れ:
クエリで対象データを作る
レポートで印刷レイアウトにする

Query

クエリとSQLの基本

条件で抽出

WHEREで必要な行だけを取り出します。

SELECT *
FROM T_売上
WHERE 売上日 >= #2026/01/01#;

集計する

GROUP BYで分類ごとの合計や件数を出します。

SELECT 商品名, Sum(金額) AS 合計金額
FROM T_売上
GROUP BY 商品名;

テーブルを結合

関連するテーブルをキーでつなぎます。

SELECT A.売上ID, B.顧客名
FROM T_売上 AS A
INNER JOIN T_顧客 AS B
ON A.顧客ID = B.顧客ID;

更新前に選択クエリで確認

UPDATEやDELETEの前に、同じ条件のSELECTで対象件数を確認します。

SELECT *
FROM T_顧客
WHERE 更新対象 = True;

CSV

CSVインポート・エクスポート

CSVをテーブルへ取り込む

DoCmd.TransferTextでCSVを取り込みます。実運用ではインポート定義を作って列型を固定すると安定します。

Sub ImportCsv(csvPath As String)
    DoCmd.TransferText _
        TransferType:=acImportDelim, _
        TableName:="T_取込", _
        FileName:=csvPath, _
        HasFieldNames:=True
End Sub

クエリ結果をCSVへ出力

集計クエリや抽出クエリをCSVに出します。

Sub ExportQueryToCsv(csvPath As String)
    DoCmd.TransferText _
        TransferType:=acExportDelim, _
        TableName:="Q_出力用", _
        FileName:=csvPath, _
        HasFieldNames:=True
End Sub

取込用の一時テーブル

CSVは直接本番テーブルに入れず、一時テーブルに取り込んで検査してから反映すると安全です。

流れ:
CSV → T_取込一時
T_取込一時をチェック
問題なければ本番テーブルへ追加

文字化け対策

CSVの文字コード、区切り文字、日付形式、先頭ゼロが消えないかを取り込み前に確認します。

確認項目:
- UTF-8 / Shift_JIS
- カンマ区切り / タブ区切り
- 日付形式
- ID列の先頭ゼロ

VBA

Access VBAの定番ルーチン

SQLを実行する

追加、更新、削除などのアクションクエリを実行します。

Sub ExecuteSql(sql As String)
    CurrentDb.Execute sql, dbFailOnError
End Sub

レコードセットを読む

SELECT結果を1行ずつ処理します。

Sub LoopRecords()
    Dim rs As DAO.Recordset
    Set rs = CurrentDb.OpenRecordset("SELECT * FROM T_顧客")

    Do Until rs.EOF
        Debug.Print rs!顧客名
        rs.MoveNext
    Loop
    rs.Close
End Sub

フォームを開く

条件を指定してフォームを開きます。

DoCmd.OpenForm "F_顧客", WhereCondition:="顧客ID=1"

レポートをPDF出力

レポートをPDFとして保存します。

DoCmd.OutputTo acOutputReport, "R_売上", acFormatPDF, "C:\temp\売上.pdf"
