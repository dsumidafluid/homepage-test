Excel Basics

Excel 基礎操作

表を壊さず、あとで集計しやすい形にするための基本操作をまとめます。まずは入力、テーブル化、並べ替え、フィルター、CSVの扱いを押さえます。

Input

入力と数式の基本

1セル1データにする

1つのセルに複数の意味を詰め込まず、日付、名前、数量、金額などを列で分けます。

良い例:
日付 | 商品名 | 数量 | 単価

見出し行を作る

1行目に列名を置くと、並べ替え、フィルター、テーブル化、CSV出力が扱いやすくなります。

見出し例:
ID, 名前, 日付, 分類, 金額

数式は=から始める

合計、差、積、商などの計算はセル参照で書くと、値を変更したときに再計算されます。

=A2+B2
=C2*D2
=SUM(E2:E10)

絶対参照を使う

コピーしても参照先を固定したいときは$を付けます。

=A2*$B$1

Table

表・並べ替え・フィルター

テーブル化する

データ範囲を選んでテーブルにすると、見出し、フィルター、集計範囲の拡張が扱いやすくなります。

挿入 → テーブル
または Ctrl + T

並べ替え

日付順、金額順、名前順などに並べ替えます。見出しを含む表では、表全体を対象にします。

データ → 並べ替え
基準列と昇順/降順を選ぶ

フィルター

条件に合う行だけを表示します。元データは消えません。

データ → フィルター
列見出しの▼から条件を選ぶ

条件付き書式

大きい値、重複、期限切れなどを色で見つけやすくします。

ホーム → 条件付き書式

CSV

CSVの扱い方

CSVは値だけの形式

CSVにはセルの色、幅、数式、シート構成は基本的に残りません。データ交換用と考えます。

残る: 文字・数値
残らない: 書式・数式・複数シート

先頭ゼロに注意

00123のようなIDは数値扱いで123になることがあります。取り込み時は文字列列として扱います。

郵便番号、社員番号、品番は文字列扱い

日付形式をそろえる

CSV出力前に日付形式をyyyy-mm-ddなどに統一すると、他ツールで読みやすくなります。

2026-06-29

余分な空白を消す

比較や集計の前に、前後空白や全角スペースを整えるとミスが減ります。

=TRIM(SUBSTITUTE(A2,"　"," "))

Check

作業前後の確認

元データを残す

加工前のシートやファイルを残しておくと、失敗しても戻せます。

raw_data
working
output

件数を確認する

取り込み前後で行数が変わっていないか確認します。

=COUNTA(A:A)

重複を確認する

ID列などで重複がないか確認します。

=COUNTIF($A:$A,A2)

空白を確認する

必須列に空白がないかフィルターで確認します。

フィルター → 空白セルだけ表示
